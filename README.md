# Getting Started

# Step 0

Install node.js

# Step 1

Install expo-cli

```
suo npm install expo-cli --global
```

See https://expo.io/learn

# Step 2

Install npm modules

```
npm install
```

# Step 3

Run it!

```
expo start
```

Go to the browser and the local dev page should load up

# Step 4

Download Expo app on Google Play or iOS

Scan QR code and the app will run
